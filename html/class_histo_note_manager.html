Detailed Description

The HistoNoteManager class HistoNoteManager est la classe principale de l'application : il n'existe qu'une seule instance de cette classe ( design pattern singleton). Cette classe permet de manipuler les différents HistoNotes ( Article, Tache, Mutimedia ) à l'aide de 3 attributs ( de type double pointeur sur HistoNotes) : articles, taches, multimedias. HistoNoteManager est le point de départ de toutes créations : c'est elle qui permet la creation des HistoNotes, et donc des Notes.
